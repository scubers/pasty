## ADDED Requirements

### Requirement: 全局设计 Token
系统必须定义一套中心化的设计 Token，包括颜色、排版和间距，并在整个应用程序（主面板和设置面板）中共享。

#### Scenario: 调色板定义
- **WHEN** 应用程序初始化设计系统
- **THEN** 它提供以下语义颜色：
  - 背景渐变：深蓝到紫色的线性渐变
  - 强调色：Teal (#2DD4BF)
  - 基础背景：#1a1a2e
  - 表面背景：用于玻璃拟态的半透明色

### Requirement: 玻璃拟态材质
系统必须提供一种可复用的视觉材质来实现玻璃拟态效果。

#### Scenario: 应用玻璃材质
- **WHEN** 视图应用玻璃材质修饰符
- **THEN** 它渲染出高质量的高斯模糊背景
- **AND** 它应用半透明的覆盖颜色
- **AND** 它应用细微的边框/描边以增加清晰度

### Requirement: 排版系统
系统必须定义一致的排版比例。

#### Scenario: 文本样式
- **WHEN** 在 UI 中显示文本
- **THEN** 它使用定义的字重（常规、中等、粗体）
- **AND** 它在适用的情况下遵循系统的动态字体大小
