## Why

跨平台剪贴板应用需要一个主要的用户界面，以便用户能够高效地访问和管理剪贴板历史记录。本变更提供了基础的主面板 UI 和核心交互模式，所有剪贴板操作都将基于此构建。

## What Changes

- **新增 macOS 应用 UI**：添加一个按需显示的浮动主面板窗口
- **全局快捷键**：实现 Cmd+Shift+V 键盘快捷键来切换面板的可见性
- **菜单栏集成**：添加菜单栏图标和"打开面板"菜单项
- **三层布局**：创建上（搜索框）、中（结果列表 + 预览）、下（页脚）布局结构
- **搜索功能**：对剪贴板项实现实时的基于文本的搜索（LIKE 匹配）
- **项预览**：在预览窗格中显示所选剪贴板项的内容（文本、图片）和元数据
- **列表导航**：支持点击项来选择和预览
- **窗口定位**：在当前屏幕的鼠标位置上方居中显示面板
- **自动聚焦**：面板出现时自动聚焦到搜索输入框
- **LSUIElement 配置**：设置应用不在 Dock 栏中显示

## Capabilities

### New Capabilities
- `main-panel`：用于剪贴板管理的主 UI 窗口，包含搜索、结果列表和预览
- `global-hotkey`：用于显示/隐藏面板的系统级键盘快捷键
- `menu-bar-integration`：用于应用访问的菜单栏图标和菜单
- `clipboard-search`：跨剪贴板历史项的实时搜索
- `clipboard-preview`：显示剪贴板项的内容和元数据

### Modified Capabilities
*(无 - 这是初始功能实现)*

## Impact

- **macOS 平台层**：添加 NSWindow、NSPanel 和相关 UI 组件
- **Core 层**：定义剪贴板项数据模型和搜索接口
- **新增 macOS 特定代码**：全局快捷键的事件监控、屏幕定位逻辑
- **依赖项**：除平台框架（Cocoa）外，预计无其他依赖
